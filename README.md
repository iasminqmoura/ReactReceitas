# App de Receitas
Este repositório contém o layout base para um aplicativo de receitas. Há duas telas, uma para o menu principal e outra para os dados da receita.

![](.github/receita.jpg)

## Tecnologias Usadas

- React Navigation
- React Vector Icons
- Expo
- Styled Components
